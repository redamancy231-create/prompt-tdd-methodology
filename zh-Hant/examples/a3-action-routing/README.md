# A3: 宣告式 vs NL 路由描述對照實驗

> **角色**: **反例案例**——如何誠實閉合一個無法繼續榨出訊號的實驗  
> **狀態**: CLOSED [E-]（實驗設計未能檢驗假設，而非假設被證偽）  
> **審查**: 10 輪（Codex GPT-5.5 ×5 + Qwen3.7-Max ×3 + 合併/對齊 ×1 + 諮詢 ×1），0 未閉合發現

---

## ⚠️ 讀前須知

A3 的屍檢結論是：**不是假設被證偽，而是實驗設計未能檢驗假設**。

- 天花板效應（GPT-5.5 在路由任務上接近完美）
- 樣本量不足（Pilot 15 cases，95% 信賴上界 ~18%）
- GT 定義爭議（"正確路由"的操作化缺乏外部錨點）
- DV 退化（二分類 accuracy 對格式效應不敏感）

A3 的**教學價值**不在於"證明了什麼"，而在於展示了一個實驗在什麼時候應該閉合、如何誠實記錄閉合原因、以及如何從"失敗"中提取方法論資產。

## 結果

### 核心資料
- 三個階段 Δ=0，discordant 率 = 0%
- Pilot 95% 信賴上界 ~18%（Rule of 3）
- 評分 bug（acceptable_paths 雙重使用 + meta action 懲罰）→ 修正後不影響結論

### 科學門: FAIL

原因不是"H1 被證偽"，而是"實驗設計未能創造檢測 H1 的條件"：
1. 天花板太高——GPT-5.5 在路由任務上接近完美
2. DV 不敏感——二分類 accuracy 無法區分格式效應
3. GT 無外部錨點——"正確路由"由實驗者定義，沒有獨立驗證

---

## 關鍵教訓

1. **不是所有實驗都能檢驗假設**：當實驗設計無法創造檢測空間時，繼續追加樣本是浪費——誠實地閉合比"做出顯著結果"更有價值
2. **屍檢比結果更重要**：A3 的價值不在 Δ=0，而在識別出 4 個根因（天花板 / DV 選擇 / GT 缺失 / underpowered）
3. **Pilot 的止損價值**：A3 v2 設計經審查發現 DV 退化——如果 v2 被執行了，浪費的是 5 倍的 API 呼叫 + 評分 + 審查時間
