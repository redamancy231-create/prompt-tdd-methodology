## 三个跨实验模式

### P1: 天花板效应是 prompt 实验的沉默杀手

**观察**: A2（代码审查）和 A3（路由决策）都在 GPT-5.5 temp=0 下遇到天花板——模型表现太好，效应量无法检测。

**机制**: 当 baseline 准确率 > 0.9 时，Δ 的检测空间被压缩到 < 0.1。需要远超常规的样本量或更粗糙的 DV 才有统计显著性——但更粗糙的 DV 本身会丢失信息。

**关键数据**: A2 correctness_rate A=0.954, B=0.935, Δ=0.019, n_nonzero=4<5（科学门 FAIL）。A3 两个实验均 Δ=0, discordant 率=0%。

**教训**: 实验设计阶段必做 ceiling probe（CK6）。如果探测显示天花板不可打破，调整 DV 或任务域——不要指望"大样本就能检测到"。

### P2: IV 污染比统计方法错误更危险

**观察**: 对照实验的核心威胁不是统计方法选择（Wilcoxon vs t-test），而是 IV（prompt 格式）被信息量差异污染。

**A2 关键发现**: B 组（prep/exec/post 分段）实际包含了 A 组（一体式编号列表）没有的检查维度——不是因为格式不同，而是因为信息量不等价。

**教训**: CK4 INVENTORY 等价协议不是可选检查——它是防止"测量了错误的东西"的唯一手段。如果两组 prompt 的产出清单不等价，实验结论无效，无论 p 值多少。

### P3: 阴性结果 ≠ 实验失败

**观察**: A2 和 A3 科学门全部 FAIL——但产出了 21 个方法论片段、3 个实验设计模板、一套可复用的分析管线。

**教训**: 阴性实验的价值不在"支持 H1"，而在：(a) 排除了特定条件下的格式效应存在性；(b) 暴露了实验设计的系统性盲点（天花板、DV 选择、IV 污染）；(c) 积累了"如何不做一个错误实验"的操作知识。

**诚实标注**: 两个实验成熟度均标 [E-]（ceiling-limited），不是 [E]——但仍然是可引用的有效证据。

## 六个缺失模式（PM-1~PM-6，来自源复盘 §4.3 对抗验证）

以下缺失模式是 A2+A3 复盘经 Codex GPT-5.5 魔鬼代言人式对抗验证后识别的——不是"实验做错了什么"，而是"系统性盲点：下一次实验应该补什么"。

| 编号 | 模式 | 定义（源报告原文） |
|------|------|---------|
| **PM-1** | 模型-温度组合作为普适混淆变量 | 两个实验均仅用 GPT-5.5 temp=0——格式效应的阴性结论不能推广到其他模型/温度。**经 Qwen 跨模型复现部分缓解**（A2 方向一致，Δ=−0.014），但审查行为/方法论传递/等价性验证等维度仍仅 GPT-5.5 单点 |
| **PM-2** | 实验者方法论成熟度作为跨实验的向上偏误 | A2 和 A3 的实验者（同一个人/团队）在两实验中方法论成熟度持续提升——A3 的设计质量高于 A2。这对"跨实验模式"声称构成混淆：观察到的改进是方法学成熟还是真有跨域规律？ |
| **PM-3** | 探索性 vs 确认性框架的持续模糊 | Tier 0 本质是探索性（生成假设），Tier 1 是确认性（检验假设）——但实验报告未明确区分两者的推断逻辑和结论语言 |
| **PM-4** | 测试集规模×效应量检测的灵敏度关系 | 未分析测试集规模和项目区分度对统计功效的联合影响——只知道"n=4 不够"，但不知道"需要多少" |
| **PM-5** | 评分/分析工具自身的实现有效性需要预审 | A3 的评分 bug（acceptable_paths 双重使用 + meta action 惩罚）在收集后才被发现——评分脚本应该和实验设计一样经过预审 |
| **PM-6** | 独立审查自身的误判风险未被充分讨论 | 复盘经 10 轮异后端审查——但审查者之间的分歧、审查衰减、审查者的系统性盲点本身未被分析。审查不是神的视角 |
