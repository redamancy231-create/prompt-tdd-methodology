# 参与贡献 · Contributing

感谢关注！这是文档/方法论项目，贡献方式与代码项目有所不同。

## 贡献方式

### 报告问题
- **勘误/笔误** → [提交 Issue](../../issues/new?template=corrigendum.yml)
- **方法论疑问** → [提交 Issue](../../issues/new?template=methodology-question.yml)
- **功能建议** → 提交 Issue（标签: enhancement）

### 提交改进
1. Fork 本仓库
2. 创建分支: `git checkout -b fix/description`
3. 修改后提交：清晰描述改了什么、为什么
4. 发起 Pull Request

### 语言
中/English 均可。涉及内容修正的 PR 请同步更新三语言文件。

## 行为准则
- 批评对准内容，不针对人
- 欢迎反例和反面证据
- 不确定时先开 Issue 再发 PR

## 许可
贡献内容默认继承本仓库 CC BY 4.0 许可。
